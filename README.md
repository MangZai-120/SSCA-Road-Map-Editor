# SSCA 思路编辑器

一个**单文件、零依赖**的可视化思维 / 树状图编辑器。右键从一个节点拖到另一个节点即可用曲线连接，左键拖空白平移画布，每个节点有大标题 + 小标题，点开在右侧编辑备注。支持白天 / 黑夜主题、多导图隔离存档、节点窗口式拖拽改大小。

---

## 🚀 最快用法（下载一个小文件，约 35 KB）

1. 下载本仓库的 **`index.html`** 这一个文件
2. 双击用浏览器打开 —— 即开即用，所有数据自动存在浏览器本地

> 不需要安装任何东西，一个 `index.html` 就是完整程序。

也可以直接访问 GitHub Pages 在线使用（启用 Pages 后）：`https://<用户名>.github.io/<仓库名>/`

---

## 🖥️ 桌面应用用法（非浏览器，无右键/手势干扰）

需要本机已安装 [Node.js](https://nodejs.org/)。

- **Windows**：双击 **`启动SSCA思路编辑器.bat`**（首次会自动安装运行环境，之后秒开）
- 或命令行：

```bash
npm install
npm start
```

桌面应用基于 Electron，是独立窗口，**不是浏览器**，所以没有浏览器右键菜单、扩展或触摸板手势。

---

## ✨ 功能

- **节点**：每个节点有大标题（名字）+ 小标题（简述），单击在右侧操作栏编辑备注
- **连线**：右键按住一个节点 → 拖到另一个节点 → 曲线连接
- **画布**：左键拖空白平移，滚轮以光标为中心缩放
- **改大小**：拖节点右边 / 下边 / 右下角，像改窗口一样调节点宽高
- **多导图隔离存档**：像「多世界存档」，每个导图独立保存，可新建 / 切换 / 重命名 / 删除
- **白天 / 黑夜主题**：黑暗模式护眼配色，记忆偏好
- **导入 / 导出**：单个导图可导出 / 导入 JSON
- 所有数据存浏览器 `localStorage`（桌面应用同理，按应用隔离）

## ⌨️ 交互一览

| 操作 | 行为 |
|---|---|
| 左键拖空白 | 平移画布 |
| 左键拖节点 | 移动节点 |
| 右键按住节点 → 拖到另一节点 | 曲线连接 |
| 单击节点 | 右侧编辑大/小标题、备注 |
| 双击空白 | 新建节点 |
| 拖节点右边/下边/右下角 | 窗口式改大小 |
| 单击连线 + Delete | 删除连线 |
| 滚轮 | 缩放 ・ Esc 取消 ・ Delete 删除选中 |
